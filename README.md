# 雅正永富｜客戶管理 CRM

禹盛開發代銷案「雅正永富」的客戶管理系統前端。

## 架構
- 前端：本 repo 的 `index.html`，部署於 GitHub Pages
- 資料來源：Google 試算表「客戶總覽」（由 Apps Script 整合 5 張表單產生）
- 線上網址：https://yushenecorp.github.io/yazhengyongfu-crm/

## 資料流
5 張 Google 表單（廣告來電 / 廣告聯繫 / 首參 / 回訪 / 電話追蹤）
→ Apps Script 整合程式
→ 客戶總覽試算表（最新狀態、完整歷程、報表事件）
→ 本前端讀取顯示

## 主要功能
- 客戶總覽：階段、業務歸屬、需求、追蹤提醒
- 待派工：未指派且未結案名單，由管理者分派
- 報表：依日期統計來電 / 來人 / 回訪
- 快速建名單：自動查重、避免重複建檔

## 維護注意
- 資料一律從表單進，客戶總覽為程式自動產生、唯讀
- 更新前端＝替換本 repo 的 `index.html` 後 commit
